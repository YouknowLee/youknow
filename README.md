실행 가이드
youknow-0.0.1-SNAPSHOT.jar를 다운받습니다.

콘솔창에서 java -jar youknow-0.0.1-SNAPSHOT.jar 를 실행합니다.
http://localhost:8080 에 접속을 하면 화면을 확인 할 수 있습니다.
http://localhost:8080/console 에 접속을 하면 DB에 접근 할 수 있습니다.
